# 渠道开放平台

# 目的和收益

1. 自动化，减少人力，提高运营效率。

    1. 很多可见数据（有价率、准备率、失败率）客户可以自行查看。

    2. 订单日志可以自行查看。

2. 遇到问题，有时差的话，客户可以自助查看。

3. OP 可以更加专注于策略以及更重要事情。

4. 服务差异化。

## 指标与仪表盘

1. 有价。

2. 准确率。错误的可以下载日志？

3. 成功率。可以下载日志查？

4. 当前 QPS 设置？（ 能不能给 ）

5. 当前超时设置? 客户可以自己更改?出现超时问题了,客户可以自行更改



## 匹配

1. 客户可以下载他们自己上传已匹配的文件等。

2. Dida 热销未匹配。

3. 帮客户 做/检查 房型/酒店 匹配。



## 日志

1. 订单的各个阶段日志。

    1. 提高 OP 的效率，相关的日志，客户可以先自行下载查看。

    2. 减少争议



## API 更新告知

API 有更新的话，时机合适，可以进行推送



## Dida 联系人以及更新

1. 负责这个账号的负责人和联系方式以免更改了负责人，他们不知情

2. CS 联系人以及更新。

3. 财务联系人以及更新。

4. BD 联系人以及更新。
